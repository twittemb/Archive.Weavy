Stitch

public struct Stitch

A Stitch is the result of the weaving action between a Warp and a Weft (See Warp.knit function) It describes the next thing that will be presented (a Presentable) and the next thing the Loom will listen to have the next navigation steps (a Weftable). If a navigation action does not have to lead to a Stitch, it is possible to have an empty Stitch
